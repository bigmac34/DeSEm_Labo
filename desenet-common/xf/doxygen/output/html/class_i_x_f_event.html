Detailed Description

Interface for an event. The abstract methods defined in this interface need to be reimplemented in its derived classes. These methods are used by other classes in the framework (ex. XFThread, XFReactive).

Requirements:

  • Needs to hold a property which allows to distinguish between different types of events.
  • Use an enumeration to define the type for the type property.
  • The type property needs to be set at initialization time and cannot be changed afterwards.
